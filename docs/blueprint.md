# **App Name**: Smart School Manager

## Core Features:

- Student Management: Add, edit, archive, and delete student records including full name, grade, date of birth, current balance, and monthly fee.
- Student Payment Tracking: Record student payments, automatically deducting monthly fees, and maintain a payment history for each student.
- Staff Management: Manage staff profiles, including positions and salaries.  Calculate salaries for hourly staff based on hours worked.
- Expense Tracking: Record and categorize school expenses with dates, amounts, and descriptions.
- Financial Overview: Display a dashboard with key metrics like total income, expenses, top debtors, and monthly financial analysis charts.
- Automated Report Generation: Use AI to automatically generate a summary report of students with balances owing. The tool will decide when or if to list those with smaller debts.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust, stability, and intelligence.
- Background color: Very light blue (#E8EAF6), almost white, for a clean and professional feel.
- Accent color: Soft green (#8BC34A) for positive actions and highlights.
- Body text: 'PT Sans', a modern sans-serif for good readability
- Headlines: 'Playfair', a modern serif, for titles, balanced with PT Sans
- Use a consistent set of educational and financial icons. Ensure they are simple, clear, and align with the color scheme.
- Maintain a clean, data-dense layout with well-organized tables. Prioritize readability and a clear visual hierarchy. Ensure the app is responsive and adapts to different screen sizes.
